- ```python
a = int(input())

### 6033_아스키문자표

- ```python
a = ord(input) # ord() 아스키 문자표의 넘버를 출력해줌
print(chr(a+1)) 문자표상의 넘버에 해당하는 문자의 다음 문자를 출력해줌.

### 6042_소숫점 반올림

```python
a = float(input())
print(format(a,".2f"))
```
`format(<float>,".nf")` 소숫점 n+1 번째 자리에서 반올림.
