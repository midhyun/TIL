#### 프로젝트 Mohobby
>Fact
- 풀스택 Engineer로 참여
- 프로젝트 기간 3주간 배포까지 구현
- Multi-Step-Form 으로 UX 개선
- 유효성 검사 개선, 캡챠 기능 추가
- Kakao Map API를 활용한 키워드 검색구현,

- UX개선은 유저에게 큰 만족감을 줄 수 있음
